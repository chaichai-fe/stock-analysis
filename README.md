# 股票分析

基于 Next.js 的股票分析应用：输入 A 股代码，查看自上市以来的**平均年化收益率（CAGR）**与**每年精确分红率**。数据来自**东方财富**网站（爬取公开接口）。

## 功能

1. **简洁界面**：仅需一个输入框输入股票代码（如 600519、000001）。
2. **真实数据**：通过东方财富 K 线与分红接口获取历史数据。
3. **整理结果**：
   - 自成立起平均年化收益率（前复权、复利年化）
   - 有分红时，按年统计并展示每年分红率（当年分红总额 / 年末股价）

## 使用

```bash
pnpm install
pnpm dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)，输入股票代码后点击「查询」。

## 股票代码说明

- 支持 **A 股 6 位代码**：沪市（如 600519、601318）、深市（如 000001、300059）。
- 可带后缀：`600519.SH`、`000001.SZ` 或 `SH600519`、`SZ000001`。

## 技术栈

- Next.js 16（App Router）
- TypeScript
- Tailwind CSS
- 东方财富公开接口（push2his K 线 + 数据中心/F10 分红）
